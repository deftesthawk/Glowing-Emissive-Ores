Instructions

There are currently 4 Minecraft Modloaders: NeoForge, Forge, Fabric, and Quilt. You will need to install different requirements depending on which Modloader you are using. I have listed all the possible combinations you can use for each Modloader in the requirements tab.

Confused? Here's an example:

I plan on playing a Forge modpack: First, I install "Glowing Emissive Ores", then test if it works (some packs come pre-installed with the requirements). If it doesn't work, then come back to this site and select the requirements tab. Next, I will choose either Option 1, 2, 3, or 4 depending on the version of Minecraft I am playing on. If I choose option 1, I will need to install both "MoreMCMeta (Forge)" and "MoreMCMeta Emissive Textures (Forge)". Finally, I will load the modpack up again and test it. For most modpacks, this should work.
